포지셔닝 1. 포지셔닝 주요 속성 1.1 box-sizing : 박스 크기의 기준 설정 - content-box : width, height의 적용기준이 content 기준 - border-box : width, height의 적용기준이 border 기준 1.2 float : 블록요소가 물에 떠있는 상태로 주변상황에 맞추어 배열 흘러가는 기준 적재되는 기준, 옆에 다른 content들은 밀리게 된다. - left | right | none 1.3 clear : float 속성을 해제 - left : 이전 float:left; 를 해제 - right : 이전 float:right; 를 해제 - both : 양쪽 모두 해제
