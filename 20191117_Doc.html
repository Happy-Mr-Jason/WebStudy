CSS 기초 1. 선태자 : 서식 적용범위 선택 1.1 전체 선택자 *{...} Ex> *{ margin : 0px; padding : 0px; } 1.2 태그 선택자 : 태그명 기준으로 서식 적용 태그명{...} Ex> p{ font-size:12px; font-family:돋움; } 1.3 클래스 선택자 : 태그 속성중 class="클래스명" 정의된 태그에 대한 서식 적용 class라는 의미 처럼 특정 단체를 지정하는 용도 .클래스명{...} Ex> .bluetext{ color : blue; } 1.4 id 선택자 : 태그 속성중 id="아이디명" 정의된 태그에 대한 서식 적용범위 id라는 의미 처럼 1개의 객체를 지정하는 용도 #아이디명{...} Ex> #container{ background:#ff6600; } 1.5 그룹 선택자(,) : 다수개의 선택자에 동일 서식을 적용 태그명, .클래스명, #아이디명, ....{...} Ex> h1, h2 { text-align:center; } 2. CSS 2.1 !important : 충돌되는 서식 중 가장 먼저 서식을 적용하고자 할 경우 서식 다음에 기술 Ex> .bluetext{ color:blue !important; } 2.2 서식 적용순서(명시도 관점) 1. 인라인스타일 2. id스타일 3. class스타일 4. 태그스타일 2.3 동률 순위이며, 충돌되는 서식일 경우 마지막 지정 서식이 적용 2.4 브라우저 접두사 - webkit : 웹킷 방식 브라우저(사파리, 크롬) - moz : 게코 방식(모질라, 파이어폭스) - o : 오페라 - ms : 마이크로소프트 인터넷 익스플로러
