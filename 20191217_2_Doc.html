텍스트관련 스타일 1. 글꼴 관련 스타일 1.1 font-family : 글꼴 1.2 font-size : 글꼴 크기 1.3 font-weight : 굵기 - normal : (400) - bold : (700) - bolder : 현재 보다 더 굵게 - lighter : 현재 보다 더 가늘게 - 100~900 : 100단위로 값으로 조정 1.4 font-variant : 대문자 표현 방식 - normal - small-caps : 작은크기의 대문자 1.5 font-style : 기울임꼴 - normal - italic - oblique 1.6 font : 속기속성 font: |caption|icon|menu|message-box|small-caps|status-bar 2. 텍스트 스타일 2.1 color : 문자 색상 2.2 text-decoration - none - underline : 밑줄 - overline : 윗줄 - line-through : 취소선 2.3 text-transform : 대소문자 변환 - none - capitalize : 첫 글자 대문자로 변환 - uppercase : 모두 대문자 - lowercase : 모두 소문자 - full-width : 전각 문자 변환 2.4 text-shadow : 텍스트에 그림자 효과 주기 <가로 거리><세로 거리><번짐 정도><색상> 2.5 white-space : 공백처리 - normal 공백과 Enter 자동 줄바꾸기 - nowrap : 1칸 무시 없음 - pre : 그대로 입력된데로 없음 - pre-line : 1칸 입력된데로 허용 - pre-wrap : 그대로 입력된데로 허용 2.6 letter-spacing : 글자 간격 조절 - normal | <크기> - word-spacing : 단어 간격 조절 - normal | <크기> 3. 문단 스타일 3.1 direction : 문자 기술 방향 - ltr : default, left-to-right - rtl : right-to-left 3.2 text-align : 문단 정렬 - start : direction 기준 시작 위치 정렬 - end : - left : - right : - center : 가운데 정렬 - justify : 양쪽 정렬 - match-parent : 부모 요소 맞추어 정렬 3.3 text-justify : text-align : justify 일 경우 단어간격이 많이 차이가 날 때 고려. 문자 사이 간격을 조절 - auto - none - inter-word : 단어 간격으로 조절 - distribute : 글자 간격으로 조절 3.4 text-indent : 첫째 줄 들여쓰기. 음수 입력시 내어쓰기 가능 3.5 line-height : 줄 간격. 문자 높이 포함. 백분율 권장 3.6 text-overflow : border안에 텍스트를 모두 표현하지 못할 경우 - clip - ellipsis
