1. 이미지 1.1 웹용 이미지 - Gif : 다양한 색상 표현이 미흡, 용량이 적다. 투명이미지 제공, 애니메이션 기능 제공 아이콘, 불릿, 배너 제작 - Jpg/Jpeg : 사진용, 실사 이미지 - Png : 투명 지원, 사진같은 화질 제공, 네트워크용으로 개발 1.2 이미지 삽입 1.2.1 속성 - src : 그리파일 경로 - alt : 이미지 설명용(표시안함), 그림 경로가 잘못되었거나 클라이언트로 전송되지 못한 경우 그림 대신에 나타날 텍스트, 시각 장애인용 텍스트 읽기 프로그램용 소스 - width : 이미지 폭 - height : 이미지 높이 1.3 : 설명글 적용 대상 지정용, 컨테이너 1.4 : 설명글 붙이기 2. 링크 만들기 2.1 하이퍼링크 설정. 텍스트 2.1.1 속성 - href : 링크할 사이트 주소 #id id='' 를 이용하여 화면 하나내에서 북마크처럼 생성해서 사용할 수 있다. - target : 링크 내용이 표시될 위치 - download : 링크용이 아닌 다운로드 용도로 사용(브라우저에 플러그인이 있으면 파일을 화면에 띄우고, 그렇지 않으면 파일이 다운로드되어짐.) - rel : 현재 문서와 링크 문서와의 관계 알려줌 - hreflang : 링크 문서의 언어 - type : 링크 문서의 파일 유형 2.1.2 CSS 선택자 a:link a normal, unvisited link a:visited a link the user has visited a:hover a link when the user mouses over it ( Must come afger a:link and a:visited) a:active a link the moment it is clicked (Must com after a:hover) 3. 이미지 맵 만들기 3.1 : 이미지 맵 요소를 묶어주고 이미지에 반영될 수 있게 함. 3.2 : 가상 링크 객체, 이미지 위에 있는 투명한 객체 3.2.1 속성 - shape : 맵의 모양 3가지(default, rect, circle, poly) - coords : shape 객체의 좌표 (시작좌표x,y,끝좌표x,y) //circle인 경우 (중심x,중심y,반지름) //rect인 경우 대각선 방향의 꼭지점의 x,y좌표 //poly인 경우 모든 꼭지점의 x,y좌표
