배경 스타일 1. 배경색과 이미지 1.1 background-color : 배경색 1.2 background-clip : 배경 적용 범위 - border-box : 테두리까지 적용 - padding-box : 테두리 제외한 페딩까지 적용 - content-box : 내용부분만 적용 1.3 background-image : 배경 이미지 - background-image:url(파일경로) 1.4 background-repeat : 배경 이미지의 반복 속성 - repeat : default - repeat-x - repeat-y - no-repeat 1.5 background-size : 배경 이미지의 크기 - auto : 원래 크기(default) - contain : 요소안에 꽉 차게 - cover : 요소안에 꽉 차게 <크기값> <백분율> 1.6 background-position background-position:<수평위치><수직위치>; - 수평위치 : left|center|right| <백분율> 또는 <길이값> - 수직위치 : left|center|right| <백분율> 또는 <길이값> 1.7 background-origin : 배경 기준 위치 - border-box : 테두리 기준 - padding-box : 페딩 기준 - content-box : 내용부분 기준 1.8 background-attachment : 스크롤시 이동 - scroll : 스크롤시 화면에서 사라짐(default) - fixed : 스크롤시 화면에서 위치 고정 1.9 background : 속기 속성
